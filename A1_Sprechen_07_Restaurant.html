🏭 Situation
🍽️
Im Restaurant
You are at a German restaurant. Order food and drink, ask for the bill.
💬 Speaking Prompts
Prompt 1
Ask for a table.
Haben Sie einen Tisch für … Personen?
Prompt 2
Order food.
Ich nehme … / Ich möchte … bitte.
Prompt 3
Order drinks.
Ich hätte gern … zu trinken.
Prompt 4
Ask for the bill.
Die Rechnung, bitte. / Kann ich zahlen?
🔑 Key Phrases
Haben Sie einen Tisch frei?Ich nehme …Ich möchte …Die Rechnung, bitte.Was empfehlen Sie?Einmal / Zweimal …Getrennt oder zusammen?Das schmeckt gut!
📚 Model Dialogue
A: Guten Abend! Haben Sie einen Tisch für zwei Personen?
B: Ja, natürlich. Hier bitte.
A: Danke! Ich möchte die Suppe und einmal Schnitzel, bitte.
B: Und zu trinken?
A: Ein Wasser und einen Apfelsaft, bitte.
B: Gern. Kommt sofort!
A: Die Rechnung, bitte.
B: Zusammen oder getrennt?
