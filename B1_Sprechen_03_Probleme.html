Describing and explaining problems — formal and informal contexts

🏭 Situation
🔧
Ein Problem lösen
You rented a flat and have issues: broken heating, noisy neighbours. Describe the problems to your landlord.
💬 Speaking Prompts
Prompt 1
State the problem clearly.
Das Problem ist, dass … / Es gibt ein Problem mit …
Prompt 2
Describe what happened.
Seit … / Als ich … bemerkte ich …
Prompt 3
Explain the impact.
Das führt dazu, dass … / Dadurch …
Prompt 4
Request a solution.
Ich bitte Sie, … zu / Könnten Sie bitte …?
🔑 Key Phrases
Das Problem ist, dass …Es gibt ein Problem mit …Seit einer Woche …Das führt dazu, dass …Ich bitte Sie, …Könnten Sie das bitte beheben?Das ist leider nicht akzeptabel.Ich erwarte eine baldige Lösung.
📚 Model Dialogue
A: Guten Tag, ich rufe wegen meiner Wohnung an.
B: Ja, was kann ich für Sie tun?
A: Das Problem ist, dass die Heizung seit drei Tagen nicht funktioniert.
B: Das tut mir leid. Haben Sie schon versucht, den Techniker zu erreichen?
A: Ja, leider ohne Erfolg. Außerdem sind die Nachbarn sehr laut, was dazu führt, dass ich nachts nicht schlafen kann.
B: Ich verstehe. Ich werde mich sofort darum kümmern.
🎯 Quick Check

Q3

Q4

Q5
